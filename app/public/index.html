API

GET /delay/{delay}
Example: /delay/50 :: all requests are delayed by 50ms
ResponseCodes: 200 OK
GET /delay/{delay}/{percentage}
Example: /delay/75/10 :: 10% of all requests are delayed by 75ms, the rest are not delayed
ResponseCodes: 200 OK
POST /auth/token
Example payload: {"user": "user1", "pass": "111"}
ResponseCodes: 200 OK or 403 Unauthorized
GET /authenticatedendpoint
Headers: Authorization: {token}
ResponseCodes: 200 OK or 403 Unauthorized
